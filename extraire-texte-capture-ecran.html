Extraire le texte d'une capture d'écran

Vous avez fait une capture d'écran et vous voulez en récupérer le texte sans le retaper ? Avec Image2OCR, il suffit de la coller. Voici comment.

Une capture d'écran est une image : le texte qu'on y voit n'est pas sélectionnable. Pour le récupérer, on utilise l'OCR (reconnaissance de texte). Image2OCR rend l'opération immédiate : pas besoin d'enregistrer puis d'importer un fichier, vous collez directement.

La méthode la plus rapide : Ctrl + V

  1. Faites votre capture d'écran (sur Windows : Win+Maj+S ; sur Mac : Cmd+Maj+4).
  2. Ouvrez Image2OCR dans votre navigateur.
  3. Collez avec Ctrl+V (ou Cmd+V sur Mac). La reconnaissance démarre toute seule.
  4. Copiez le texte obtenu, ou exportez-le (TXT, Word…).
